== Arbitrary section ==

1. Open Amazon Settings from Admin Sidebar of your website and Choose Product Type and Add Affiliate tag of your amazon affiliate account.
2. Download this plugin and goto Dashboard. Open Amazon Settings and copy the Site Key.
3. Register on https://importwoocommerce.com
4. After registration and logging in; open the "My Profile" from the top menu
5. Paste that Site key in the Target Website Key Field and Save it. (It will show you the Website name where your products will be added)
6. Now Open "Get Amazon Data" page and paste Search URL OR Category URL or Single Product URL from amazon.com and Press Fetch

It will start getting the products from those links to your website
